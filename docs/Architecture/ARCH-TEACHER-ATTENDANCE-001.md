# ARCH-TEACHER-ATTENDANCE-001 先生出席

## Reality

- `07_出席ログ.teacher_id` は会員出席の受付・確認主体であり、先生自身の出席回数には使用しない。
- `11_先生マスタ` は `member_id` を任意で関連付ける。先生就任後も会員ID、級・段位は独立して維持する。
- 先生出席の事実は `16_先生出席ログ` に保存する。
- 一回は `稽古日 + slot_id + teacher_id` で数える。
- 担当区分は `主先生` または `副先生` とする。
- 給金対象・単価・金額は、運用確認前には判定・保存しない。
- 稽古日と課金枠の曜日が一致しない登録は拒否する。

## 初版の画面責務

- 仮想ログイン中の先生を初期表示する。
- 道場・課金枠をQRまたはURLパラメータから取得する。
- 稽古枠と担当区分を選択し、登録・選択解除できる。
- 月単位で主先生、副先生、合計の出席回数を表示する。
