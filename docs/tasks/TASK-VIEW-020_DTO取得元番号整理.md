# TASK-VIEW-020 DTO取得元番号整理

## Purpose

`20_会費状態View`は複数シートの集約であり、DTO項目の取得元をコード上で追跡しやすくする。

## Rule

内部DTO変数を `s<シート番号>_<意味>` とする。

- `s01_*`: `01_会員マスタ`
- `s04_*`: `04_月次選択`
- `s05_*`: `05_請求明細`
- `s06_*`: `06_入金ログ`
- `s07_*`: `07_出席ログ`
- `s09_*`: `09_決済エビデンス`
- `s20_*`: View内で導出した値

## Compatibility

スプレッドシートの既存ヘッダーとWeb APIレスポンスは変更しない。番号付けはView行を生成する内部DTOへ限定する。
