# TASK-DEV-022 タイムトリップ設定・システム時刻表示

STATUS: 実装済・画面スモーク確認待ち

## 目的

画面操作による月次Story確認で、GASとブラウザが同じテスト日時・対象月を使用していることを利用者へ明示する。

## 実装

- スプレッドシートの`道場管理`メニューへ`テスト時刻を設定`を追加する。
- 設定変更はスプレッドシートのモーダルだけに限定し、公開Web APIには更新入口を設けない。
- 公開画面は`system_context`を読取り、システム日時・対象月・タイムトリップ状態を表示する。
- 出席日、会費受付日、会費状況の初期期間をシステム日時へ連動する。
- 無効化時は`DEBUG_TARGET_MONTH`を適用せず、実日時・実月へ戻す。

## 完了条件

- `STORY-902 第3層 画面スモーク`のSmoke 1とSmoke 5がPASSする。
- テスト中は黄色の警告が全対象画面に表示される。
